# Flappy Bird Clone (Python & Pygame)

Đây là một bản sao đơn giản của trò chơi nổi tiếng **Flappy Bird**, được viết bằng ngôn ngữ Python và thư viện Pygame. Dự án này được thực hiện trong tuần đầu tiên của khóa học lập trình.

## 🎮 Cách chơi
- Sử dụng phím **Space** (Dấu cách) để giúp chú chim bay lên.
- Tránh va chạm với các đường ống màu xanh.
- Nếu chim rơi xuống đất hoặc va vào ống, trò chơi sẽ kết thúc.
- Nhấn **Space** sau khi Game Over để bắt đầu lại.

## 🚀 Cài đặt & Khởi chạy

### Yêu cầu
- Đã cài đặt Python 3.x
- Đã cài đặt thư viện Pygame

### Cách chạy
1. Cài đặt Pygame (nếu chưa có):
   ```bash
   pip install pygame
   ```
2. Chạy file game:
   ```bash
   python FB.py
   ```

## 📂 Cấu trúc dự án
- `FB.py`: Mã nguồn chính của trò chơi.
- `BG2.png`: Hình nền của trò chơi.
- `FB2.png`: Hình ảnh chú chim.
